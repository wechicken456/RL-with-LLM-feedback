# RL-with-LLM-feedback

This repo contains scripts that aim to improve reinforcement learning (RL) by giving the agent extra artificial guidance rewards generated by an LLM. These rewards are not meant to replace real environment reward, but to supplement it to shape learning and accelerate convergence.

This project currently uses the following packages:
- Environments and RL API: [gymansium](https://gymnasium.farama.org/).
- RL algorithms: [CleanRL](https://github.com/vwxyzjn/cleanrl)
- OpenAI API. 

Currently, there's only implementation of DQN with the [Taxi-v3](https://gymnasium.farama.org/environments/toy_text/taxi/) environment.
